## Core Technologies

The project leverages several foundational technologies that form the backbone of the game's architecture and functionality.

### Game Engine and Framework
- **Unity 6000.4 LTS**: Primary game development platform providing cross-platform deployment capabilities
- **Netcode for GameObjects (NGO v2.11.0)**: Advanced networking solution enabling real-time multiplayer experiences
- **Universal Render Pipeline**: High-performance graphics rendering system supporting modern visual effects

### AI and Intelligence Systems
- **Behavior Designer**: Professional-grade AI behavior tree framework for sophisticated NPC behavior
- **Unity AI Assistant**: AI-powered development assistance tools and patterns
- **Hybrid FSM-Behavior Tree Architecture**: Advanced AI decision-making combining finite state machines with behavior trees

### Communication and Integration
- **websocket-sharp (v1.0.3-rc11)**: WebSocket library enabling external AI agent control
- **Unity Transport Layer**: Underlying networking infrastructure for reliable client-server communication
